# Оно тебе надо 📦🔨

Учебный проект — веб-страница интерактивного онлайн-аукциона необычных лотов, выполненная в рамках курса «Веб-разработчик» от **Яндекс Практикума**.

---

## 🛠 Стек технологий

* **HTML5:** семантическая разметка, форма обратной связи, встроенное видео.
* **CSS3:** Flexbox, CSS Grid, позиционирование элементов, работа с пользовательскими шрифтами и фоновыми изображениями.
* **БЭМ (BEM Nested):** методология именования и организации файлов проекта.

---

## 🚀 Особенности проекта

* Соблюдение правил семантической вёрстки для улучшения доступности и SEO.
* Реализация верстки карточек лотов с адаптивными сетками (CSS Grid).
* Работа с декоративными элементами, наложением слоев (z-index) и псевдоклассами.
